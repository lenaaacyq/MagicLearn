# EduPal｜平行世界伴侣（英语学习 Agent Demo）

核心定位：不是“做题工具”，而是“平行世界伴侣”。  
你可以把背单词/练口语包装成“剧情任务线”，用多模态开场解决冷启动，用长期记忆与任务规划体现 Agent，而不只是 Chatbot。

## 本地运行

```bash
npm i
npm run dev
```

## 这个 Demo 展示什么

- **多模态开场**：支持粘贴视频 URL 作为“开场视频”，用于模拟 Runway/Pika/Sora 或国产视频生成的产物
- **情感化语音**：用浏览器的 Web Speech API 播放 TTS（用于 Demo），后续可替换为 ElevenLabs 或国产 TTS（科大讯飞/火山/百度等）
- **Agent 的核心**：
  - **长期记忆**：本地 `localStorage` 记录卡点单词、上次情绪、完成任务
  - **任务规划**：按学习阶段自动生成“寻找秘宝”的任务线，并用进度条可视化

## 下一步接入建议（优先国产模型）

- 文本大模型：阿里通义千问 / 智谱 GLM / 百度文心 / 讯飞星火
- 语音识别（ASR）：讯飞/百度/火山/腾讯
- 语音合成（TTS）：讯飞/火山/百度（可做多音色/情感）
- 视频生成：可对接国产视频生成或第三方工具的产出链接

## Roadmap（MVP → Beta → V1）

| 阶段 | 目标 | 交付物 | 验收标准 |
| --- | --- | --- | --- |
